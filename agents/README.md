# Cabbits Agents

This folder defines the intended agent system for Cabbits.

The visible experience is the Companion Agent. Other agents are internal services that support the companion.

## Agents
- companion
- planner
- evaluator
- teacher-math
- teacher-writing
- safety
- memory

## Principle
The companion is the product. Specialist agents should feel like abilities of the companion, not separate apps.
