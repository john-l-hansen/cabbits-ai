# Philosophy

The companion comes first.

Learning is collaborative.
Curiosity is more valuable than speed.
Mistakes are signals.
Rewards reinforce effort rather than addiction.
Knowledge should feel like discovery.
The interface should disappear.
The user should feel accompanied rather than instructed.
The companion should remember growth, not just facts.

Cabbits should feel like a soft echo: alive enough to return to, quiet enough to trust.
