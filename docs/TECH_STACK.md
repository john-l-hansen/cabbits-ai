# Tech Stack

## Initial Foundation
- Next.js
- TypeScript
- Tailwind
- App Router

## Later
- Supabase for auth, database, and memory
- Vercel for deployment
- Motion or Rive for expressive animation
- AI provider integration after the first UI loop is stable

## Current Rule
No database, auth, AI calls, payments, multiplayer, or advanced orchestration in the first scaffold.
