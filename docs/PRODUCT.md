# Product Definition

## Audience
Kids, teens, and young adults.

## Primary Use
Mobile-first learning sessions with desktop support for deeper study, projects, and group challenges.

## Product Pillars
1. Persistent AI companion
2. Adaptive learning paths
3. Specialist teacher sub-agents
4. Game-like progression
5. Minimal interface with expressive animation
6. Individual and group challenges

## Success
The user feels guided, challenged, rewarded, and understood over time.
