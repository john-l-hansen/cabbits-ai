# Cabbits

Cabbits is an adaptive learning companion web app.

## Start

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Current Scope

This scaffold includes:
- Next.js App Router foundation
- Mobile-first home screen
- Companion creation route
- First quest preview route
- Source documentation
- Agent instructions

No database, auth, AI calls, payments, or multiplayer are included yet.
